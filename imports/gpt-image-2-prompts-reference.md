# GPT-Image-2 提示词参考（与《纸偶》分镜配套）

> 关键地址（必须保留）
>
> https://github.com/EvoLinkAI/awesome-gpt-image-2-prompts/tree/main

## 用途
- 作为《纸偶》分镜转画面提示词（Prompt）的主参考库。
- 用于后续视频生产阶段（镜头拆解 -> 画面生成 -> 连贯性修正）。

## 使用约定（简版）
1. 先以分镜表为准（镜号/景别/机位/动作/情绪）。
2. 再从上述仓库挑对应的 prompt 模板。
3. 每个镜头输出两版：
   - 稳健版（优先一致性）
   - 创意版（优先表现力）
4. 统一保留以下字段，便于批量生成：
   - 镜号
   - 角色
   - 场景
   - 景别
   - 机位
   - 光线/色调
   - 风格关键词
   - 负面约束（避免穿帮）
