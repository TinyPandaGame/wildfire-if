# 沈夜（Shen Ye）— Character Lock

> 角色锁是 scene-agnostic：跨任意镜头/场景，沈夜的外形必须保持一致。
> 任何含沈夜的 shot prompt 都要把本文档的视觉字段嵌入。

## 基础

- **年龄**：15 岁少年
- **身高**：173cm
- **体型**：偏瘦但爆发力强；劳动型爆发体态（**不是健身房肌肉，不是运动员体型**）
- **背景**：来自衰败工业海港"铁港"，父亲已故，自己在码头搬货 2 年。在地下篮球场"狗笼"打球。

## 脸

- **脸型**：清瘦削方下颌，颧骨明显
- **五官**：眉骨硬朗，眼神锐利冷硬，鼻梁挺直，薄嘴唇
- **表情基线**：少笑，沉默，警觉。被生活磨硬过早的劳动者眼神。
- **肤色**：晒得偏黑（长期码头日晒）

## 发

- **发型**：短寸（自己剪的，不规整）
- **发色**：纯黑

## 服装

- **上身**：洗得发白起球的灰色旧背心
- **下身**：廉价深色长裤
- **鞋**：旧球鞋，右脚趾处有破洞

## 标志性细节

- 前臂多处旧伤划痕
- 手掌新裂口（搬货留下，结痂中）
- 手指有裂口
- 劳动者的手——粗糙、有茧、不修指甲

## 气质 / 姿态

- 沉默、克制、警觉
- 劳动者的姿态，不是模特或运动员的站姿
- 永远是"过早被磨硬"的少年的存在感

## 负面约束（永远不要出现）

- gym muscle / 健身房肌肉
- anime / cartoon style / 动漫风格 / K-pop idol styling
- smiling / 笑容 / 阳光开朗的表情
- polished glamour / 光鲜亮丽 / 整洁干净的衣服
- model-like beauty / 模特脸 / 奶油小生
