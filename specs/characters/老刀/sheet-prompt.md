# 老刀 — 三视图定型表 Prompt

> **用途**：给生图 agent / GPT-Image-2 / Midjourney / 即梦 等模型生成老刀的角色定型表（三视图）。
> **输出格式**：单张图 1920×1080（16:9），横向并排三视图（front/side/back）。
> **输出路径**：候选丢 `assets/art/characters/老刀/sheet-candidates/`，定稿放 `sheet.png` + `ref.png`。
>
> Agent 任选中文版或英文版（信息等价）发给模型即可。

---

## 中文版（推荐发给即梦 / 国内模型）

```
【角色定型表 — 老刀（Lao Dao）— 三视图】

主体：
58-63 岁中国男性，前职业球员，因伤退役后长期在铁港地下球场“狗笼”出没。
身高约 185cm，瘦长干硬体型，筋骨感明显；不是病弱老人。

脸型：
窄长脸，颧骨与下颌线清晰。

五官：
眼窝略深，眼神锐利，鼻梁较直，唇薄。
表情基线：冷静审视，少表情，像在读对手动作。

肤色：
长期户外风吹日晒的暗麦色，带粗糙岁月纹理。

发型：
短发偏乱，发量减少，花白（黑白混）。

服装：
旧夹克/旧外套或深色旧T，深色旧长裤，旧运动鞋或旧工装鞋。
整体“实用、旧、耐操”，不要时装化。

标志性细节：
常叼一根未点燃的烟；
右腿轻跛（姿态和受力可见微弱代偿）；
手掌明显偏大，指节粗，长期训练痕迹明显。

气质：
安静但压场，懂球老手，不靠吼叫建立威慑。

【输出格式 — 必须严格遵守】
单张图，1920×1080，16:9 横屏。
横向并排三视图，从左到右依次为：
  1. 正面（Front view）— 全身正对镜头
  2. 侧面（Side view）— 左侧 90° 全身
  3. 背面（Back view）— 全身背对镜头
姿势：自然站立，双臂略外展，不要夸张摆拍。
可轻微体现右腿轻跛的重心差，但不能夸张成残疾化。

【背景】
纯净浅灰中性背景，无场景、无道具、无地面阴影。
柔和均匀光照，不要戏剧化强反差。

【风格】
二次元美漫融合风（Anime × American comic hybrid）
+ 写实电影感骨架（cinematic realism）
+ 赛璐璐线条清晰度 + 美漫墨线质感
+ 粗粝但克制的角色定型呈现。

【NEGATIVE — 严禁出现】
fashion old gentleman look, polished studio portrait,
frail sick elder, exaggerated villain scars, cartoon stylization,
young middle-aged replacement, no limp clue,
multiple separate characters, scene background,
3/4 angle view (must be strictly front / side / back ONLY),
extra limbs, deformed hands, finger count errors,
face drift between the three views
```

## 英文版（推荐发给 GPT-Image-2 / Midjourney / Sora 等）

```
[CHARACTER SHEET — Lao Dao / 老刀 — three-view turnaround]

SUBJECT:
A 58–63-year-old Chinese male, former professional basketball player,
forced to retire due to injury, now a seasoned underground-court mentor figure.
About 185cm tall. Lean, long, dry-athletic frame with visible tendon/bone structure.
Not frail, not soft.

FACE:
Narrow long face, defined cheekbones and jawline.

FEATURES:
Slightly deep-set sharp eyes, straight nose, thinner lips.
Default expression is calm, analytical, and severe.

SKIN:
Dark wheat / sun-weathered tone with rough age texture.

HAIR:
Short, slightly messy, thinning, salt-and-pepper to mostly gray.

CLOTHING:
Worn jacket/outerwear or old dark T-shirt,
dark worn pants, old athletic or work shoes.
Practical and weathered, not stylish.

DISTINCTIVE MARKS:
Unlit cigarette often held in mouth,
slight limp on right leg,
unusually large palms with thick knuckles and training wear.

MOOD/POSTURE:
Quiet authority, veteran observer energy.
No loud posturing; intimidation comes from presence.

[OUTPUT FORMAT — STRICTLY ENFORCED]
Single image, 1920×1080, 16:9 horizontal.
Three full-body views in one frame, left to right:
front / left-side 90° / back.
Natural standing pose, arms slightly away from torso.
Subtle right-leg compensation is allowed but not exaggerated.

[BACKGROUND]
Plain light-gray neutral background, no props, no environment,
no dramatic shadows, soft even lighting.

[STYLE]
Anime × American comic hybrid on cinematic realism foundation,
cel-shaded line clarity + US-comic ink texture,
gritty but neutral character-sheet finish.

[NEGATIVE]
fashionable old-gentleman styling, polished glamour portrait,
frail sick old man, cartoon exaggeration,
young middle-aged replacement look,
missing right-leg limp cue, missing oversized-hand cue,
multiple characters, scene background,
3/4 angle (front/side/back ONLY),
extra limbs, deformed hands, finger count errors,
identity drift across views
```

## 操作流程

1. 使用“最终用版”先跑 3–5 次（同 prompt 不同 seed）
2. 候选图保存到 `assets/art/characters/老刀/sheet-candidates/`
   命名：`YYYY-MM-DD-attempt-NN.png`
3. 提交后等待审核，选定后复制为：
   - `assets/art/characters/老刀/sheet.png`
   - `assets/art/characters/老刀/ref.png`
4. 后续含老刀镜头（shot-17/18）统一引用 `ref.png`
