# 《野火》Story Bible v2

## 一句话概括
铁港码头长大的野球少年沈夜，从地下斗球场一路杀入GSL，最终披上国家队战袍站上世界杯——一部关于"野性"如何在体制内存活并燃烧的篮球史诗。

---

## 世界观设定

### 架空体育体系
故事设定在一个**架空现代世界**，国家和地理虚构，但社会结构、科技水平与现实相当。

### 联赛金字塔

| 层级 | 名称 | 性质 |
|------|------|------|
| **T0** | **🏆 FIBA世界杯（World Crown）** | 四年一届，国家队对决，篮球世界的终极荣耀。冠军国家授予"王冠之星"称号。沈夜的最终目标不是俱乐部冠军，是披上国家队战袍站在这里 |
| **T0** | **🏅 环球运动会（The Universals）** | 四年一届（与世界杯错开两年），综合体育盛会中的篮球项目。金牌含金量不亚于世界杯，但赛程更残酷（短周期高密度） |
| **T1** | **GSL（Global Supreme League）** | 全球顶级俱乐部联赛，12支超级俱乐部，跨洲主客场制。年度联赛+季后赛，篮球商业化的巅峰，球员薪资天花板 |
| **T1** | **洲际国家杯（Nations Crown）** | 两年一届，各大洲国家队争夺洲际霸权。世界杯的预演和练兵场，也是新人证明自己配得上国家队的舞台 |
| **T2** | **大洲旗舰联赛** | 各大洲最高职业联赛（西洲SWL、北洲NBL、东洲EPL等），GSL选材池。能在这里打首发就已经是一国顶尖 |
| **T2** | **洲际冠军杯（CCL）** | 各洲俱乐部冠军对决，一年一届淘汰赛。俱乐部通往GSL的跳板 |
| **T3** | **NDL（National Dynasty League）** | 国内顶级职业联赛，20支队伍，升降级制 |
| **T4** | **NDL二级联赛（NDL-2）** | 半职业，NDL的升降级通道 |
| **T5** | **城市联赛 / 青训杯** | 各城市业余联赛+俱乐部青训梯队赛 |
| **T6** | **地下 / 街头** | 无组织野球、地下赌球场、街头锦标赛 |

**两条上升路线：**
- **俱乐部路线**：街头→城市联赛→NDL-2→NDL→大洲联赛→GSL（挣钱、荣誉、个人巅峰）
- **国家队路线**：熔炉计划→国家队征召→洲际国家杯→世界杯/环球运动会（为国而战、终极荣耀）
- 两条线**不冲突但会撕裂**——俱乐部赛季和国家队窗口期冲突，伤病管理、忠诚度选择都是戏剧张力

### 🔥 蓝色监狱式核心机制：「熔炉计划」

NDL为了打破国家队在洲际赛上的长期低迷，联合体育总局启动了一个极端人才选拔项目——**「熔炉」（The Crucible）**。

**规则：**
- 从全国海选 **200名** 25岁以下潜力球员（职业、半职业、野球均可入选）
- 关入封闭训练基地，进行 **多阶段淘汰赛**
- 每阶段设定不同赛制（1v1、3v3、5v5、混编对抗、生存赛等）
- **输了就出局，没有复活赛**（但后期有"复仇者通道"——被淘汰者有一次挑战机会）
- 最终存活的球员获得：NDL优先签约权 + 国家队征召资格 + 海外联赛推荐
- 每个阶段有 **数据评估系统「鹰眼」**，实时量化球员能力值（速度/力量/技巧/视野/心理/团队）

**熔炉不是一次性的**——它每年举办，是贯穿故事的长期机制。沈夜第一次参加被淘汰，第二次杀回来，后期甚至成为熔炉的"传说级毕业生"。

## 主角

### 沈夜（Shen Ye）
- **年龄：** 15岁（故事开始，初中毕业）
- **身高：** 173cm（初始）→ 持续发育，各阶段身高变化是重要剧情线
  - 铁港篇：173cm（被嘲笑"这身高打什么篮球"）
  - 熔炉·一：178cm（开始窜个，但仍是全场最矮之一）
  - NDL-2：183cm（身高追上来了，但对抗还是吃亏）
  - 熔炉·二：187cm
  - NDL：190cm（终于不算矮了，但对位锋线仍有差距）
  - 海外/GSL：195-198cm（最终身高，配合逆天运动能力成为错位噩梦）
- **位置：** 初期无固定位置（什么都不会），身高变化导致位置不断调整——矮的时候被迫打后卫，长高后逐渐能打锋卫摇摆，最终成为无视位置的全能球员
- **出身：** 铁港区码头，父亲是码头工人（已故），母亲在鱼市摆摊
- **外号：** "野狗"（铁港街头）→"野火"（熔炉时期）→ 后期由对手和媒体赋予新外号

**身体素质（核心设定）：**
沈夜是一个**纯粹的身体天赋怪物**，篮球技术几乎为零：
- 🔥 **弹跳力：** 逆天级，190cm的身高能做出2米+级别的扣篮，码头搬货练出的腿部爆发力
- 🔥 **体力：** 无底洞，从小在码头干重活+跑酷，全场飞奔40分钟不带喘的
- 🔥 **速度/反应：** 百米级冲刺能力，街头混大的动物性反应速度
- 🔥 **对抗：** 190cm但身体硬度像内线，撞不动
- ❌ **投篮：** 完全不会，姿势都是错的
- ❌ **运球：** 只会最基础的，一加速就掉球
- ❌ **传球：** 没有传球意识，脑子里只有自己上
- ❌ **战术：** 零，不懂挡拆、不懂跑位、不懂轮转防守
- ❌ **规则：** 前期连走步和三秒违例都搞不清

**→ 类比：樱木花道的运动天赋 + 排球少年日向翔阳的"什么都不会但身体素质碾压"**

初期纯靠身体素质横冲直撞，能抢篮板、能追防快攻、能暴扣，但一打正规比赛就漏洞百出。被所有人视为"只是个运动员，不是篮球手"。

**能力解锁树（随剧情逐步习得）：**

| 阶段 | 解锁能力 | 契机 |
|------|----------|------|
| 铁港篇 | 篮板嗅觉 + 暴力扣篮 | 天生的，街头野球唯一需要的 |
| 熔炉·一 | 基础运球 + 上篮终结 | 被正规球员吊打后痛定思痛 |
| 熔炉·一 | 防守脚步 | 韩教练的地狱特训 |
| NDL-2 | 中距离投射 | 苦练半年，命中率从20%→45% |
| 熔炉·二 | 传球视野觉醒 | 和陆澄磨合后理解"传球能得分" |
| NDL | 三分球 | 从不敢投到关键时刻敢出手 |
| NDL | 挡拆战术 | 真正理解团队篮球 |
| CCL | 阅读比赛 | 面对外洲战术体系被迫进化 |
| 海外 | 节奏控制 | 从"永远全速"到"知道什么时候该慢" |
| GSL | 全能整合 | 所有能力融为一体，创造自己的篮球风格 |

**每次解锁都不是突然开窍，而是大量训练+实战失败+关键领悟的结果。**

**性格：**
- 单纯直接，脑子不复杂，行动派
- 不服输到近乎偏执——被打脸100次也要站起来第101次
- 对"体制内的人"有天然敌意（底层出身的防御机制）
- 重情义但不善表达
- 初期有点像愣头青，后期沉稳但骨子里的野性不灭

**核心成长弧线：**
运动白痴（只有身体）→ 学会基本功 → 被迫合作 → 理解团队 → 技术全面化 → 领袖 → 在世界舞台定义自己的篮球

## 重要配角

### 队友线

**陆澄（Lu Cheng）** — "机器"
- 190cm，纯控卫，青训体系出身，数据流打法
- 和沈夜是完全对立的风格——理性vs野性
- 从互相看不顺眼到最佳搭档
- 后期：沈夜最信任的人，也是最了解他弱点的人

**姜铁（Jiang Tie）** — "塔"
- 206cm，中锋，铁港老乡，沈夜从小的球搭子
- 天赋平平但意志力惊人
- 作用：沈夜与"过去"的纽带，提醒他不要忘本

**莫星河（Mo Xinghe）** — "鬼手"
- 185cm，得分后卫，左撇子，投射天才
- 性格轻浮花哨但关键时刻极度冷血
- 后期成为沈夜在海外联赛的对手

### 对手线

**顾北辰（Gu Beichen）** — "王座"
- 195cm，全能锋卫摇摆，篮球世家出身
- NDL最年轻MVP，被称为"天选之子"
- 沈夜的终极宿敌——天才vs野兽
- 不是反派，是真正的强者，互相成就

**赤城凛（Akashiro Rin）** — 西洲联赛
- 混血球员，在海外联赛等待沈夜的墙壁级对手
- 代表"体系化篮球的极致"

**"幽灵"（The Phantom）**
- GSL传说级控卫，沈夜的终极目标
- 前期只通过录像和传闻出现，后期正面对决

### 导师线

**刀叔（老刀）**
- 铁港地下球场的老头，前职业球员，因伤退役后沉沦
- 教会沈夜基本功，也是沈夜篮球的启蒙者
- 在沈夜进入熔炉前病故——成为沈夜的精神驱动力

**韩教练（韩鹤）**
- 熔炉计划的总教官，冷酷理性
- 看到沈夜的天赋但不偏袒，甚至故意打压
- 后期揭示：他年轻时也是野球出身，和沈夜是同类人

## 篇章结构（长篇连载）

### 第一篇：铁港（街头篇）
**约3-5卷**
- 铁港码头的地下球场生态
- 沈夜在街头称王的过程
- 城市野球锦标赛——被球探发现
- 刀叔的死，沈夜决定"走出铁港"
- **篇末钩子：** 熔炉计划的邀请函

### 第二篇：熔炉·第一季（淘汰篇）
**约5-8卷**
- 200人封闭训练，阶段淘汰
- 1v1斗牛赛 → 3v3街头规则 → 5v5正式赛 → 混编生存赛
- 沈夜和陆澄的冲突与磨合
- 与顾北辰的第一次正面交锋（沈夜惨败）
- **沈夜在第四阶段被淘汰**——不是主角光环一路赢
- 篇末：沈夜加入NDL-2球队，从底层重新开始

### 第三篇：NDL-2（磨砺篇）
**约3-5卷**
- 半职业联赛的残酷现实（低薪、差旅、无人关注）
- 沈夜磨练投射和战术素养
- 帮助球队冲击升级赛
- **复仇者通道开启**——沈夜杀回熔炉

### 第四篇：熔炉·第二季（复仇篇）
**约5-8卷**
- 规则升级，更残酷的淘汰机制
- 新一批怪物级新人加入
- 沈夜已经不是当初的野狗，蜕变后的打法让所有人震惊
- 和顾北辰的第二次对决（势均力敌）
- **沈夜最终存活，获得NDL签约权**

### 第五篇：NDL（职业篇）
**约8-12卷**
- 加入NDL弱队，从保级开始
- 菜鸟赛季的挣扎（体能、伤病、媒体压力）
- 逐渐成为球队核心
- 季后赛之路
- 和顾北辰的第三次对决（NDL总决赛）
- **篇末：** 洲际冠军杯资格到手

### 第六篇：洲际冠军杯（CCL篇）
**约5-8卷**
- 第一次面对外洲球队
- 不同风格的篮球碰撞（力量型、技术型、跑轰型）
- 赤城凛登场
- 沈夜开眼界但也暴露短板
- **结果：** 止步四强，但引起海外球探注意

### 第七篇：海外联赛（异乡篇）
**约5-8卷**
- 转会到西洲SWL联赛
- 语言、文化、战术体系全面冲击
- 和赤城凛同联赛对抗
- 莫星河也在海外，亦敌亦友
- 从替补打到首发，证明自己

### 第八篇：国家队（荣耀篇）
**约3-5卷**
- 国家队征召，和熔炉旧友重聚
- 世界杯/洲际国家队赛事
- 沈夜+顾北辰+陆澄同队——曾经的对手成为队友
- 打到决赛但惜败——为GSL埋下伏笔

### 第九篇：GSL（至高殿堂）
**约8-12卷**
- 被GSL球队签下
- 世界最强的12支球队
- "幽灵"终于正面出场
- 从新秀到全明星
- GSL总决赛——这不是终点，是新的起点

### 第十篇及以后：（开放式）
- GSL卫冕/三连冠挑战
- 新一代挑战者出现
- 沈夜从球员到传奇的转变
- 可以一直写下去

## 蓝色监狱式穿插机制

### 1. 能力量化系统「鹰眼」
每个球员有**十二维数据雷达图**，分为四大类：

**身体素质（Physical）**
- **速度（SPD）** — 冲刺、变向、第一步启动
- **力量（PWR）** — 身体对抗、卡位、背打
- **弹跳（JMP）** — 垂直弹跳、滞空、二次起跳
- **体力（STA）** — 持久力、攻防转换、第四节体能

**进攻技术（Offense）**
- **运球（DRB）** — 控球、突破、护球
- **中投（MID）** — 急停跳投、抛射、中距离命中率
- **三分（3PT）** — 定点、接球就投、后撤步三分
- **传球（PAS）** — 精度、创造力、助攻视野

**防守与篮板（Defense）**
- **防守（DEF）** — 脚步、协防、抢断、盖帽
- **篮板（REB）** — 卡位意识、进攻篮板、长篮板判断

**精神属性（Mental）**
- **心理（MND）** — 关键球表现、抗压、领导力
- **野性（WLD）** — 不可预测性、创造力、打破常规的能力（沈夜的核心属性）

数据不是静态的——比赛中会实时波动，读者可以直观看到角色的成长。

**沈夜初始数据（15岁/173cm）：**
| 属性 | 值 | 说明 |
|------|-----|------|
| SPD | A | 百米级 |
| PWR | B+ | 码头练出来的 |
| JMP | S | 逆天 |
| STA | A+ | 无底洞 |
| DRB | F | 几乎不会 |
| MID | F | 不存在 |
| 3PT | F | 不存在 |
| PAS | F | 没有意识 |
| DEF | D | 有本能但没技术 |
| REB | B+ | 靠弹跳硬抢 |
| MND | C | 热血但不稳定 |
| WLD | A | 天生的 |

→ 一眼就能看出来：身体素质炸裂，技术全是F，典型的"运动员但不是篮球手"。

### 2. 淘汰赛制的持续紧张感
不只是熔炉——整个故事充满淘汰机制：
- 熔炉的阶段淘汰
- NDL-2的升降级（输了就留在地狱）
- NDL的保级战
- CCL的单场淘汰赛
- 海外联赛的合同年压力（表现不好就被裁）
- GSL的赛季制

### 3. 对手觉醒 / 进化时刻
参考蓝色监狱的"觉醒"设定：
- 每个重要对手都有一个 **"极限状态"**——在极端压力下触发的超常发挥
- 沈夜的极限状态叫 **"野火"**——进入一种本能接管理性的zone
- 不是超能力，是对"心流状态"的戏剧化表现
- 每次触发条件不同，不是想开就能开

### 4. 回忆杀与平行叙事
- 重要比赛中穿插对手的背景故事（蓝色监狱式"这个人为什么这么强"）
- 不同角色的视角章节
- 铁港的回忆作为沈夜的精神锚点反复出现

### 5. 赛间篇 / 日常篇
- 训练日常、更衣室文化、球员社交
- 伤病康复（不回避体育的残酷面）
- 商业代言、媒体采访（成名后的烦恼）
- 感情线（克制处理，不喧宾夺主）

## 主题

- **野性 vs 秩序：** 天赋和自由能否在体制中存活？
- **出身与命运：** 底层少年能走多远？
- **孤独与信任：** 独狼如何学会依靠别人？
- **传承：** 刀叔 → 沈夜 → 未来的新人

## 风格基调

- 热血但不幼稚，比赛描写注重战术细节和身体感受
- 对话风格偏硬派，少废话
- 篮球专业性要高，但用叙事包裹，不掉书袋
- 暗线伏笔多，角色在几十章后才回收的铺垫

---

## 写作节奏

- 每卷 5-8 章，每章 3000-5000 字
- 比赛篇和日常篇交替
- 每篇结尾必须有强钩子（新对手登场 / 重大转折 / 角色危机）
- 适合连载更新的节奏：每章有小高潮，每卷有大高潮
