# 场景设定 — Episode 1

## 1. 铁港码头（西侧）
- 集装箱堆得像积木塔。探照灯惨白，黄昏短促
- 沈夜签字位置：货车后方夹板
- 海风咸腥，手指有裂口被风吹疼
- 视觉风味词：脏橘色、惨白切片、海风、汽笛

## 2. 锈街（码头→狗笼路径）
- 修车铺卷帘门、早餐摊蒸笼、光膀子大叔喝啤酒
- 时间晚于 19:00，全街进入'收摊时段'
- 巷口有人喊'三组打二组'——熟人社交密度高

## 3. 狗笼（地下球场）
- 废弃集装箱仓库改装。铁皮门挂歪嘎吱声
- 顶上漏天，下雨能没脚踝；今晚干燥
- 应急灯昏黄从码头偷的；铁丝网围一圈
- 篮筐：生锈铁框，扣下去螺丝会松
- 水泥地直接落地，膝盖钉钉子的痛感
- 看客 ~10–15 人：码头工、锈街混混、同龄少年
- 空气：汗+烟+海腥

## 4. 筒子楼四层（沈夜家）
- 两室一厅，'厅'=过道+桌+两椅
- 水泥墙腻子脱落一半；厨房排风扇坏 3 个月没修
- 窗户关不严，海风灌进来吹晾晒衣服
- 厨房灯亮着是母亲未睡的标志
- 沈夜房间：床+桌，桌上摊课本翻开页一字未写

## 视觉锚点（占位阶段保留）
- 探照灯白
- 狗笼的铁丝网构图
- 起跳前的水泥地视角
- 厨房盖盘子的鸡蛋面
- 卧室天花板低视角（'再长几厘米能伸手够到'）
